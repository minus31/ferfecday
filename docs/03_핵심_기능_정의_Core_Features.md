# 3장. 핵심 기능 정의 (Core Features)
## 3.1 인텔리전트 다이내믹 택일 엔진
- 전수 조사 연산: 입력된 3~5일간의 기간 내 모든 날짜와 12개 시간대의 사주팔자(총 3660개 조합)를 자동 생성하여 엔진에 대입합니다.
- 하이브리드 용신 판정: 단순 자평 평점 방식에서 나아가, 조후(기후 균형)와 억부(일간 강약 균형) 조건문을 통합 적용한 최적의 용신(用神) 글자를 도출합니다.
- 대운 평점 보정: 원국이 다소 한쪽으로 치우쳐 있더라도 초년청년기(제1~3대운)가 용신·희신 방향으로 흐르면 대폭 가산점을 부여하는 동적 랭킹 시스템을 구동합니다.
## 3.2 개인 맞춤형 프리미엄 리포트 (The 프리미엄 리포트)
- 성향 진단: 오행 기도 비율과 일주론을 결합하여 아이의 성품 기본 틀을 제시합니다.
- 성공과 재능: 식재관(食財官)의 연결 흐름을 추적하여 학업 머리, 재물 복, 직업적 대성 가능성을 도출하고 직무 적성을 추천합니다.
- 건강 및 액막이 처방전: 취약 오행에 기반한 소화기/기관지 예방 건강 진단 및 흉살(삼형살, 원진귀문, 공망 등)을 놀이 치료와 전문 직업군으로 상쇄하는 업상대체 가이드를 제시합니다.-> 유형분류 후 DB화 필요
