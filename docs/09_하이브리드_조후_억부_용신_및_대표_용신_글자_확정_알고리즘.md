# 9장. 하이브리드 조후·억부 용신 및 대표 용신 글자 확정 알고리즘
사주의 기후적 편중을 살피는 조후(調候) 평가를 1차로 수행한 뒤, 이상이 없을 경우 억부(抑扶)를 적용하는 시스템 판정 트리입니다.
## 9.1 하이브리드 조후 판정 트리 (Decision Tree)
- 조후 불균형 조건식 정의:
- 7장에서 산출된 오행별 최종 기도 산출 결과에서, 아래 두 조건 중 하나에 해당되면, 조후 균형 붕괴로 정의합니다.
(1)화 또는 수 중 하나만 존재(나머지 하나는 0%) 하고, 그 기도 비율이 20% 이상인 경우,
(2)두 오행 모두 존재할 때는 강한 오행의 기도비율이 20% 이상이고 and 약:강 기도비율이 1:3 이상인 경우,
조후 균형 붕괴로 정의합니다.
- 조후 불균형 판정 로직:
```python
# [개정된 하이브리드 조후 판정 엔진 알고리즘]

def calculate_hybrid_yongshin(saju_elements, wol_ji):
    # 1. 7장에서 계산된 최종 오행 기도 값 추출 (0.0 ~ 100.0)
    pct_fire = saju_elements.get('fire_pct', 0.0)
    pct_water = saju_elements.get('water_pct', 0.0)

    # 2. 조후 불균형 여부 판정 (제시해주신 정밀 조건 적용)
    is_johu_collapsed = False
    dominant_element = None # 사주 전체를 너무 뜨겁거나 차갑게 만드는 주범 오행

    # (조건 1) 화 또는 수 중 하나만 존재할 때
    if (pct_fire > 0 and pct_water == 0) or (pct_water > 0 and pct_fire == 0):
        active_pct = max(pct_fire, pct_water)
        if active_pct >= 20.0:
            is_johu_collapsed = True
            dominant_element = 'Fire' if pct_fire > 0 else 'Water'

    # (조건 2) 둘 다 존재할 때 (약:강 비율이 1:3 이상으로 치우쳤을 때)
    elif pct_fire > 0 and pct_water > 0:
        stronger_pct = max(pct_fire, pct_water)
        weaker_pct = min(pct_fire, pct_water)

        if stronger_pct >= 20.0 and (weaker_pct / stronger_pct) <= (1.0 / 3.0):
            is_johu_collapsed = True
            dominant_element = 'Fire' if pct_fire > pct_water else 'Water'

    # 3. 조후 및 억부 최종 용신 확정 트리
    if is_johu_collapsed:
        # 사주 전체가 최종적으로 '불(火)'로 치우쳐진 경우 -> 무조건 물(水)로 조율
        if dominant_element == 'Fire':
            johu_status = "조열 (Imbalance - Hot)"
            final_yongshin_element = '水'

        # 사주 전체가 최종적으로 '물(水)'로 치우쳐진 경우 -> 무조건 불(火)로 조율
        elif dominant_element == 'Water':
            johu_status = "한랭 (Imbalance - Cold)"
            final_yongshin_element = '火'
    else:
        # [중요] 조후가 중화된 사주는 월지가 여름/겨울이더라도 조후용신을 쓰지 않고 '억부용신'으로 결정!
        johu_status = "중화 (조후 양호)"
        final_yongshin_element = calculate_eokbu_yongshin()

# 아래의 세부 연산 실행
def calculate_eokbu_yongshin():
    # 1. 8장에서 구한 원국 오행 강약 지수(SI)를 기준으로 신강/신약 판단
    # 2. 신강 사주 (SI >= 0) -> 일간을 제어하는 극설(식상, 재성, 관성) 오행 중,
    #    7.1/7.2장에서 계산된 오행별 '최종 기도 비율'이 가장 큰 오행을 용신으로 채택.
    if SI_Final >= 0.0:
        candidate_scores = {
            '식상': gido_scores['식상'],
            '재성': gido_scores['재성'],
            '관성': gido_scores['관성']
        }
     # 가장 세력이 건강한(점유율이 높은) 극설 오행을 용신 오행으로 리턴
        return max(candidate_scores, key=candidate_scores.get)

    # 3. 신약 사주 (SI < 0) -> 일간을 돕는 생조(인성, 비겁) 오행 중,
    #    7.1/7.2장에서 계산된 오행별 '최종 기도 비율'이 가장 큰 오행을 용신으로 채택.
    else:
        candidate_scores = {
            '인성': gido_scores['인성'],
            '비겁': gido_scores['비겁']
        }
        # 가장 세력이 건강한(점유율이 높은) 생조 오행을 용신 오행으로 리턴
        return max(candidate_scores, key=candidate_scores.get)

    return {
        "johu_status": johu_status,
        "final_yongshin_element": final_yongshin_element
    }

```
## 9.2 대표 용신 글자(Yongshin Character) 확정 알고리즘
용신 오행(예: '水')이 결정되면, 7장(7.1 및 7.2)에서 이미 산출이 완료된 개별 글자의 최종 기도값을 비교하여 가장 강력한 단 하나의 대표 용신 글자를 확정합니다.

### 1) 후보 글자 풀(Pool) 수집 및 비교 프로세스
- 천간 후보군: 사주 원국 천간 3자리(연간, 월간, 시간) 중 용신 오행과 일치하는 글자 및 최종 기도값 호출.
- 지장간 후보군: 사주 원국 지지 4자리(연지, 월지, 일지, 시지)의 지장간 속 글자 중 용신 오행과 일치하는 글자 및 최종 기도값 호출.
### 2) 대표 용신 글자 확정 룰 (Selection Rule)
Final_Yongshin_Char=Max(후보 글자들의 7.1/7.2 최종 기도값)
- [대표 용신 글자 동점 판정 규칙 (Yongshin Character Tie-breaker)]:
용신 오행에 속하는 후보 글자들 간의 최종 오행 기도값(7.1/7.2 결과)이 완벽히 동일할 경우, 9.2절 엔진은 다음의 정통 명리 위계 순서에 따라 단 하나의 대표 글자를 최종 용신 캐릭터로 확정합니다.
- 사주의 사령탑이자 심장인 월지(Month Branch)에 가장 가깝게 밀착 배치된 자리의 글자 우선 채택: 월지 > 일지 > 시지 > 월간 > 시간 > 연간 > 연지 순.
- 기운의 활동성과 추진력이 강력한 양간(陽) 글자 우선 채택: 음양 중 음간(陰) 대비 양간을 가산 우대함. (예: 양수인 壬水 > 음수인 癸水)

- 무장(없는 오행) 사주일 때의 예외 처리 보완 룰
만약, 최종 판정된 용신 오행이 택일 후보 사주의 원국(지장간 포함)에 단 한 글자도 존재하지 않는 극단적인 경우(무장 사주):
- 보완 룰 A (희신 채택): 용신을 생조하는 희신(喜神) 오행의 가장 강건한 글자를 대신 용신 글자로 확정합니다. (예: 水가 없을 경우 水의 원천이 되는(인성에 해당하는) 金 기운 글자를 찾아 수원을 열어준다고 해석 매칭)
- 보완 룰 B (대운 희망 매칭): 용신오행의 인성에 해당하는 글자도 없다면, 리포트에는 원국에 없는 아쉬움을 대운의 유입 시기와 연결하여 긍정의 메시지를 자동 출력합니다. (예: "원국에는 수호 기운인 [수]가 보이지 않지만, 아이의 [X]세 대운부터 풍부하게 채워져 인생을 주도하게 됩니다.")
