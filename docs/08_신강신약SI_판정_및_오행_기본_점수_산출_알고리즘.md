# 8장. 신강신약(SI) 판정 및 오행 기본 점수 산출 알고리즘
산출된 목·화·토·금·수의 최종 기도를 아울러 일간을 돕는 기운(생조)과 억제하는 기운(극설)의 평형 상태를 통해 신강·신약을 수학적으로 정의합니다.
## 8.1 세력 합산 공식 및 오행 강약 지수 (Saju Strength Index, SI)
- 생조기운(생조): 일간과 동일한 비겁의 합산 기도와 일간을 생하는 인성의 기도 합. (비겁은 기운의 주체성이 크므로 가중치 1.1 적용)
생조기운=인성 기도 총합+(비겁 기도 총합×1.1)
- 극설기운(극설): 일간의 기운을 빼앗고 제어하는 식상, 재성, 관성의 기도 합. (일간을 직접 지배하는 관성은 가중치 1.1 적용)
극설기운=식상 기도 총합+재성 기도 총합+(관성 기도 총합×1.1)
- 오행 강약 지수 (SI): 생조와 극설의 편차 비율을 퍼센트(
%)로 수치화합니다. 극설기운 대비 생조기운이 얼마나 강한지를 측정함.
SI = { (생조기운−극설기운) / (생조기운+극설기운) } ×100

## 8.2 신강신약 구간 판정 및 기본 오행 평점 (Base_Score) 공식
- 기본 평점 공식: 최적의 조화 상태인 **약신강 설정 구간 중간점 (SI=17.5%)**에서 멀어질수록 감점 계수(K)를 곱하여 기본 점수를 하락시킵니다. (최종 계산 시 소수점 둘째 자리 반올림)

Base_Score = 100 −∣17.5−SI∣×K
