# 길일 점수 기준

현재 길일 점수는 `app/api/lucky-days/route.ts`의 `scoreSaju()`에서 계산한다.

이 기준은 최종 명리학 규칙이 아니라, 주어진 기간 내 날짜/시간 후보를 정렬하기 위한 1차 휴리스틱이다. 추후 실제 기준표가 확정되면 `scoreSaju()`를 교체하거나 보강한다.

## 계산 흐름

1. 사용자가 선택한 기간의 모든 날짜를 순회한다.
2. 각 날짜마다 12개 시간대 후보를 계산한다.
   - 00:00 자시
   - 02:00 축시
   - 04:00 인시
   - 06:00 묘시
   - 08:00 진시
   - 10:00 사시
   - 12:00 오시
   - 14:00 미시
   - 16:00 신시
   - 18:00 유시
   - 20:00 술시
   - 22:00 해시
3. `@orrery/core`의 `calculateSaju()`로 각 후보의 사주 정보를 계산한다.
4. `scoreSaju()`로 길일 점수를 산정한다.
5. 점수 내림차순으로 정렬하고 Top 10만 반환한다.

## 기본 점수

- 모든 후보는 `55점`에서 시작한다.
- 각 기준의 가점/감점을 더한다.
- 최종 점수는 `0~100` 범위로 제한한다.

## 기준표

| 항목 | 기준 | 점수 |
| --- | --- | ---: |
| 오행 균형 | 7장 문서 기준의 오행 기도 비율을 20% 균형점과 비교 | 최대 `+22` |
| 오행 결핍 | 기도값이 0인 오행이 있는 경우 | 결핍 1개당 `-5` |
| 좋은 간지 관계 | `合`, `六合`, `三合`, `方合` | 1개당 `+3` |
| 나쁜 간지 관계 | `沖`, `刑`, `破`, `害`, `怨嗔`, `鬼門` | 1개당 `-4` |
| 좋은 12운성 | `長生`, `冠帶`, `建祿`, `帝旺`, `養` | 1개당 `+3` |
| 약한 12운성 | `死`, `墓`, `絶` | 1개당 `-2` |
| 십성 다양성 | 천간/지지 십성 종류 수 | 최대 `+8` |
| 공망 | 공망에 해당하는 주 | 1개당 `-4` |

## 길성/신살 기준

| 항목 | 점수 |
| --- | ---: |
| 천을귀인 `cheonul` | 개당 `+5` |
| 천덕 `cheonduk` | 개당 `+4` |
| 월덕 `wolduk` | 개당 `+4` |
| 문창 `munchang` | 개당 `+3` |
| 금여 `geumyeo` | 개당 `+2` |
| 각 주의 12신살이 `將星`, `攀鞍`, `驛馬` | 개당 `+2` |
| 백호 `baekho` | `-5` |
| 괴강 `goegang` | `-3` |

## 예시

`2026-06-16 22:00` 후보가 96점으로 계산된 예:

```text
기본점수 55
오행 균형 +8
간지 관계 +2
12운성 +0
길성/신살 +24
십성 다양성 +7
공망 +0
= 96점
```

## 현재 한계

- 성별은 현재 계산 API에서 임시로 `M`으로 고정되어 있다.
- 출생지는 기본값을 사용한다.
- 2시간 단위 후보만 평가하며 분 단위 후보는 평가하지 않는다.
- 일주별 해석 텍스트는 아직 placeholder 상태다.
- 점수 기준은 서비스 검증용 휴리스틱이며 전문가 검수 기준이 아니다.

## 오행 기도 계산 기준

오행 균형 점수와 상세 화면의 `나의 오행·기도` 그래프는 `lib/saju/element-qi.ts`의 `calculateElementQi()`를 사용한다.

- 일간은 기준점으로 보고 천간 기도 계산에서 제외한다.
- 연간, 월간, 시간은 기본 기도값 `0.7`에서 시작한다.
- 지지는 지장간 월률 비율로 분해한다.
- 월지/일지/연지/시지 통근, 좌하 득세/극/설기, 인접 천간 충/합화/합반을 반영한다.
- 월지 사령, 일지 안착, 지지 정면 충, 삼합/반합 왕지 가중, 생지/고지 변질을 반영한다.
- 최종 기도 총합을 목/화/토/금/수 퍼센트로 정규화한다.
